# 📊 HR Analytics Dashboard

## 📌 Overview

This project presents an **HR Analytics Dashboard** built using Power BI to analyze employee performance, workforce trends, and organizational insights.

The dashboard focuses on transforming raw HR data into actionable insights that help businesses:

* Improve employee performance
* Optimize workforce management
* Support data-driven HR decisions

---

## 🖼️ Dashboard Preview

![HR Dashboard](hr_dashboard_preview.png)

---

## 🎯 Key Objectives

* Analyze employee performance across departments
* Identify factors affecting employee productivity
* Explore relationships between:

  * Performance & Overtime
  * Performance & Training
  * Salary & Attrition
* Support decision-making for promotion and workforce planning

---

## 📂 Dataset Description

The dataset simulates a real-world HR system and includes:

* **Employee Information**: Age, Job Role, Department, Job Level
* **Performance Data**: Rating, Training Hours, Overtime Hours
* **HR Metrics**: Promotion Flag, Attrition (Exit)
* **Business Context**: Store, Manager

---

## 📊 Key Metrics (KPIs)

* **Average Performance Rating**
* **Headcount**
* **Attrition Rate**
* **Promotion Rate**
* **Average Salary**

---

## 📈 Dashboard Features

### 1. Employee Performance Analysis

* Performance distribution across employees
* Average performance by department
* Identification of top performers

### 2. Workforce Insights

* Headcount overview
* Department comparison
* Workforce structure by job level

### 3. Performance Relationships

* Performance vs Overtime (burnout analysis)
* Performance vs Training (learning effectiveness)

### 4. Attrition Analysis

* Attrition rate by performance rating
* Attrition vs salary insights

### 5. Promotion Analysis

* Promotion rate by performance
* Identification of high-potential employees

---

## 🧠 Key Insights

* Employees with **high overtime but low performance** may indicate burnout risk
* **Training hours show positive correlation** with performance improvement
* Employees with **lower salary tend to have higher attrition rates**
* High-performing employees are more likely to be promoted
